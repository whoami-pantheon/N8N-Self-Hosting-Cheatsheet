# N8N Creation Cheatsheet

This repository contains a **cheatsheet** for setting up and securing an **N8N** instance using Docker and Docker Compose.  
It walks through container setup, permission fixes, reverse proxy configuration with **NGINX**, and SSL certificate acquisition using **Certbot**.

## About N8N

[N8N](https://n8n.io) is a workflow automation tool that lets you connect different services and build complex automations with a visual editor.  
This cheatsheet helps you self-host N8N securely and efficiently.
